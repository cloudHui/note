# Linux 命令操作审计（添加 / 查询 / 回溯）

  

> 适用：SSH 登录后的交互式 shell（root 等）  
> 日志文件：`/var/log/cmd_audit.log`  
> 配置文件：`/etc/profile.d/cmd_audit.sh`

  

---
## 1. 添加命令日志（一次性配置）

### 1.1 创建日志文件

```bash
touch /var/log/cmd_audit.log
chmod 600 /var/log/cmd_audit.log
chown root:root /var/log/cmd_audit.log
```
### 1.2 写入审计配置

```bash

cat > /etc/profile.d/cmd_audit.sh << 'EOF'

# 命令操作审计：记录时间/用户/IP/目录/命令
export HISTTIMEFORMAT='%F %T '
export HISTSIZE=50000
export HISTFILESIZE=50000
shopt -s histappend 2>/dev/null
_cmd_audit_log() {
    local last_cmd
    last_cmd=$(history 1 2>/dev/null | sed 's/^[ ]*[0-9]\+[ ]*//')
    [ -z "$last_cmd" ] && return
    echo "$(date '+%F %T') user=${USER} ip=${SSH_CLIENT%% *} pwd=$(pwd) cmd=${last_cmd}" >> /var/log/cmd_audit.log
}
PROMPT_COMMAND="_cmd_audit_log; history -a; ${PROMPT_COMMAND}"
EOF
chmod 644 /etc/profile.d/cmd_audit.sh
```
### 1.3 让配置生效
```bash
# 方式 A：退出后重新 SSH 登录（推荐）
exit
# 方式 B：当前窗口立即加载
source /etc/profile.d/cmd_audit.sh
```
### 1.4 验证是否生效
```bash
echo cmd_audit_test
tail -5 /var/log/cmd_audit.log
```
正常应看到类似：
```text
2026-07-21 10:00:01 user=root ip=10.3.112.30 pwd=/root cmd=echo cmd_audit_test
```
### 1.5（可选）日志轮转
```bash
cat > /etc/logrotate.d/cmd_audit << 'EOF'
/var/log/cmd_audit.log {
    weekly
    rotate 12
    compress
    missingok
    notifempty
    copytruncate
}
EOF
```

## 2. 查命令审计日志
### 2.1 日志格式说明
```text
时间  user=谁  ip=来源IP  pwd=当时目录  cmd=执行的命令
```
### 2.2 常用查询
```bash
# 最近 50 条
tail -50 /var/log/cmd_audit.log
# 按日期
grep '2026-07-21' /var/log/cmd_audit.log
# 查改时间相关命令
grep -E 'date -s|hwclock|timedatectl' /var/log/cmd_audit.log
# 按用户
grep 'user=root' /var/log/cmd_audit.log
# 按来源 IP
grep 'ip=10.3.112.30' /var/log/cmd_audit.log
# 分页查看整天
grep '2026-07-21' /var/log/cmd_audit.log | less
```
---
## 3. 配置生效前的历史命令怎么查
> `/var/log/cmd_audit.log` 只记录**配置生效之后**的操作。  
> 更早的操作用下面方式补充排查。
```bash
# bash 历史（无可靠时间戳，可被清理）
grep -E 'date -s|hwclock|timedatectl' /root/.bash_history
# 登录记录
last -F | head -20
# 当前系统时间
date
```
### 本次事故相关示例
```bash
date -s "2026-07-21 16:04:00" && hwclock --systohc
```  
--- 
## 4. 注意事项 
1. 只对 **SSH 交互式终端** 生效；已打开的旧窗口需重登或 `source`。
2. `cron` / 脚本非交互执行 **不会** 写入该日志。
3. 清 `.bash_history` 不影响 `/var/log/cmd_audit.log`（除非有人直接删日志文件）。
4. 以后回溯「谁改了时间」，优先查： 
```bash
grep -E 'date -s|hwclock|timedatectl' /var/log/cmd_audit.log
```
