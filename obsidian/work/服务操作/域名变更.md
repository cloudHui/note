## X-UI HTTPS 证书配置记录

  ### 当前配置

  - 域名：`www.cloudhui.cc.cd`
  - 服务器 IP：`3.112.193.21`
  - X-UI 端口：`9563`
  - 证书签发机构：Let's Encrypt
  - 自动续期工具：`acme.sh`
  - 证书路径：
    - `/root/cert/ip/fullchain.pem`
    - `/root/cert/ip/privkey.pem`

  ### 整体流程

  1. 在 DNSHE 添加 DNS 解析：

     - 类型：`A`
     - 主机名：`www`
     - 内容：`3.112.193.21`

  2. 服务器防火墙开放：

     - `80/tcp`：申请和续期证书
     - `9563/tcp`：访问 X-UI 面板

  3. `acme.sh` 联系 Let’s Encrypt，通过 80 端口验证域名。

  4. 验证成功后下载证书并安装到 X-UI。

  5. 自动重启 X-UI，使新证书生效。

  6. `acme.sh` 每天检查，并在证书到期前自动续期。

  ### 手动续期命令

  ```bash
  sudo /root/.acme.sh/acme.sh --renew \
    -d www.cloudhui.cc.cd --ecc --force

  ### 检查证书任务

  sudo /root/.acme.sh/acme.sh --list

  ### 注意事项

  - 不要删除或修改域名的 A 记录。
  - 服务器 IP 改变后，需要同步修改 DNS。
  - TCP 80 必须保持开放，否则自动续期可能失败。
  - 如果 DNS 平台提供代理或 CDN，申请证书时选择“仅 DNS”。
  - 不要移动或删除 X-UI 使用的证书文件。
  - 不要重新启用旧 IP 证书，以免覆盖域名证书。
